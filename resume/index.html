RésuméStudio

A no-install résumé editor. Your data stays in your browser — nothing is uploaded.

  1. Click any text to edit it — fields highlight on hover; edits autosave locally.
  2. Add or remove sections, entries, bullets, and skills with the dashed + and ✕ buttons.
  3. Save JSON to keep a copy, Load to bring one back, Print / PDF to export.
  4. Set an Accent color and switch Classic / Modern style anytime.
