# Personal Finance Tracker - Prototype

Your comprehensive personal finance tracking web application by Vi

---

## Complete Feature List

### Phase 1: Foundation
- **Dashboard** with financial overview
- **Monthly Overview** with 12-month tracking
- **Credit Cards** management
- **Loans** tracking
- **Subscriptions** management  
- **Calendar** with payment reminders
- **Import/Export** JSON data
- **Undo/Redo** functionality (Ctrl+Z / Ctrl+Y)
- **Light/Dark Mode**
- **Year Navigation**

### Phase 2: Intelligence
- **Auto-Sync** between all views
- **Edit Subscriptions** with full CRUD
- **Custom Categories** (unlimited)
- **Budget Management** with AI recommendations
- **Visual Indicators** for auto-calculated fields

### Phase 3: Polish
- **Enhanced Charts** with Chart.js
- **CSV Export** for all data
- **Upcoming Payments** dashboard widget
- **Smooth Animations** and transitions
- **Visual Polish** throughout
